Detailed Description

Abstract class describing a population of models.

This class should implement the following functionalities:

  • Initialize the population: usually this use a random number generator. The actual implementation is left to the derived classes.
  • Selection: decide which members of the population should be cross-overed. The default behavior is to select parents with a probability that is linear with the rank.
  • Cross-over: implements the logic based on which a child is constructed from its parents.
  • Mutation: implements the logic based on which a child is altered through random mutations.

Derive from this class by implementing at least these three methods:

  • doInitialize(): performs the initialization of the population
  • doCrossOver(): performs the cross-over.
  • doMutate(): performs the mutation of a model. Additionally, one might override the selectParents() method to change its default behavior.
